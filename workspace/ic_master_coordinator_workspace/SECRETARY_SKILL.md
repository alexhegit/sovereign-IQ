# SOVEREIGN-IQ 秘书技能 (Secretary Skill)

## 技能名称
`secretary-supplement` - 秘书情报自动补充技能

## 功能定位
当人类上传底稿后，秘书Agent接收人类指令，自动从多个外部数据源抓取补充信息，并进行多源交叉验证。

## 触发机制
**人类发指令触发**（A方案，更可控）

## 工作流程

```
人类上传底稿 → project_ingestor.py → 共享库
       ↓
人类给秘书发指令
       ↓
【秘书自动执行】
1. 输入底稿关键数据（用于交叉验证）
2. 多源数据抓取：
   ├── 企查查 → 工商信息
   └── Tavily → 新闻资讯
       ↓
3. 多源交叉验证
       ↓
4. 展示验证结果 + 冲突AI提醒
       ↓
5. 人类确认（以底稿为准）
       ↓
6. 向量化后写入共享库
```

## 多源数据源

| 数据源 | 用途 | 可信度 |
|--------|------|--------|
| **人类底稿** | 基准数据 | ⭐⭐⭐ 最高（以它为准） |
| 企查查 | 工商信息、股东结构、融资历史 | ⭐⭐ 高 |
| Tavily | 新闻、融资动态、行业信息 | ⭐ 中 |

## 关键验证字段

用于多源交叉验证的关键字段：
- `valuation` - 估值
- `funding_amount` - 融资金额
- `company_name` - 公司名称
- `team_size` - 团队规模
- `established_date` - 成立日期
- `revenue` - 营收

|--------------|------|------|

| `news_supplement` | 新闻资讯 | Tavily API |
