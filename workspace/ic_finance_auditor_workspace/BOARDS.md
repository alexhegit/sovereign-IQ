# BOARDS.md - IC Finance Auditor Dashboard

## 阶段估值框架

### 🌱 Pre-Seed / Seed 阶段

**核心估值模型：** Berkus Method, Scorecard Valuation

**适用条件：**
- 仅有创意或原型
- 无收入或早期收入 < ¥10 万
- 主要验证团队和 problem-solution fit

**Berkus 估值组件：**
| 组件 | 估值范围 | 评估要点 |
|------|-------|------|
| 团队 (Team) | ¥0-200 万 | 创始人背景、行业经验、执行力 |
| 创意 (Idea) | ¥0-100 万 | 市场痛点、解决方案创新性 |
| 原型 (Prototype) | ¥0-100 万 | MVP 完成度、用户测试反馈 |
| 战略关系 | ¥0-50 万 | 合作伙伴、渠道资源 |
| 产品推出 (Product Launch) | ¥0-50 万 | 上线计划、市场进入策略 |

**典型估值：** ¥200-800 万 pre-money

**Scorecard 权重：**
- 团队：30-40%
- 市场规模：20-25%
- 产品/技术：15-20%
- 竞争环境：10-15%
- 营销关系：5-10%
- 其他：5%

---

### 🌿 Angel 阶段

**核心估值模型：** VC Method, Early Stage Scorecard

**适用条件：**
- 有早期用户/收入
- 月收入 ¥10-100 万
- 初步验证产品市场匹配

**VC Method 公式：**
```
Post-Money = Expected Exit Value / Expected Return Multiple
Pre-Money = Post-Money - Investment Amount

典型回报倍数：10-20x (早期)
期望持有期：5-7 年
期望退出：¥5-20 亿
```

**关键指标：**
- MAU/DAU 增长
- 早期单位经济信号
- 留存率初步数据
- 市场验证进展

**典型估值：** ¥800-3000 万 pre-money

---

### 🌳 Series A 阶段

**核心估值模型：** DCF + Comparables, VC Method

**适用条件：**
- 规模化收入验证
- 商业模式可复制
- 收入 ¥100 万 -500 万 ARR

**Series A 关键指标：**
| 指标 | 优秀标准 | 健康标准 | 警示信号 |
|------|----------|---------|------|
| ARR 增长 | >100% YoY | 50-100% | <50% |
| Gross Margin | >75% | 60-75% | <60% |
| CAC Payback | <12 月 | 12-18 月 | >18 月 |
| NDR | >120% | 110-120% | <110% |
| Burn Multiple | <1.0x | 1.0-1.5x | >2.0x |

**SaaS 估值倍数：**
| ARR 规模 | P/S 倍数 | 示例 |
|-------|-------|----|
| ¥100-300 万 | 8-12x | ¥200 万 ARR → ¥1600-2400 万 |
| ¥300-500 万 | 6-10x | ¥400 万 ARR → ¥2400-4000 万 |
| ¥500 万 -1000 万 | 5-8x | ¥800 万 ARR → ¥4000-6400 万 |

**典型估值：** ¥3000 万 -1.5 亿

---

### 🌲 Series B 阶段

**核心估值模型：** DCF + Late Stage Comps, P/S Ratio

**适用条件：**
- 成熟运营指标
- 单位经济模型证明
- 收入 ¥500 万 -2000 万 ARR

**Series B 关键指标：**
| 指标 | 优秀标准 | 健康标准 | 警示信号 |
|------|----------|---------|------|
| ARR 增长 | >80% YoY | 50-80% | <30% |
| Gross Margin | >80% | 70-80% | <65% |
| Contribution Margin | >25% | 10-25% | <0% |
| CAC Payback | <10 月 | 10-15 月 | >20 月 |
| NDR | >130% | 115-130% | <110% |
| Rule of 40 | >50 | 40-50 | <35 |

**SaaS 估值倍数 (增长期)：**
| 增长率 | P/S 倍数 | 示例 |
|-------|-------|----|
| >80% | 8-12x | ¥1000 万 ARR → ¥8000 万 -1.2 亿 |
| 50-80% | 6-10x | ¥1000 万 ARR → ¥6000 万 -1 亿 |
| 30-50% | 4-7x | ¥1000 万 ARR → ¥4000 万 -7000 万 |

**典型估值：** ¥1.5 亿 -5 亿

---

### 🌲🌲 Series C+ 阶段

**核心估值模型：** DCF, P/E (如盈利), Public Comparables

**适用条件：**
- 接近市场主导地位
- 盈利路径清晰
- 收入 ¥2000 万 ARR+

**Series C+ 关键指标：**
| 指标 | IPO 标准 | 优秀表现 | 健康表现 |
|------|-------|------|------|
| ARR 规模 | >¥5000 万 | >¥1 亿 | >¥2000 万 |
| EBITDA Margin | >15% | >20% | 10-15% |
| FCF Conversion | >80% | >100% | 50-80% |
| Market Position | Top 3 | #1-2 | #3-5 |
| Revenue Quality | >90% recurring | >95% | 80-90% |

**成熟期估值倍数：**
| 指标 | 倍数范围 | 示例 |
|------|-------|----|
| P/E (盈利公司) | 15-30x | ¥3000 万 Earnings → ¥4.5-9 亿 |
| P/S (接近盈利) | 4-8x | ¥5000 万 ARR → ¥2-4 亿 |
| EV/EBITDA | 10-20x | ¥2000 万 EBITDA → ¥2-4 亿 |
| P/FCF | 20-40x | ¥1000 万 FCF → ¥2-4 亿 |

**典型估值：** ¥5 亿 - 数十亿

## 🎯 业务类型差异化估值矩阵

### SaaS / 软件服务

**核心指标体系：**
- **ARR/MRR**：核心估值锚点
- **NDR**：>120% 优秀，<100% 风险
- **Gross Margin**：应 >75-85%
- **CAC Payback**：<12 月健康
- **Rule of 40**：Growth + Profit >40%
- **LTV/CAC**：>3x 健康

**估值方法：**
```
Early Stage (ARR < ¥300 万): P/S = 8-15x
Growth Stage (ARR ¥300 万 -¥2000 万): P/S = 5-10x
Mature Stage (ARR > ¥2000 万): P/S = 4-8x
```

**调整因子：**
- NDR > 120%: +20-30%
- NDR 100-120%: 0%
- NDR < 100%: -20-40%
- Gross Margin > 80%: +10-20%
- Gross Margin < 70%: -10-20%

---

### 电商 / 零售

**核心指标体系：**
- **GMV**：规模指标
- **Take Rate**：平台抽佣 10-30%
- **Repeat Rate**：>30% 健康
- **Contribution Margin**：单位经济
- **Inventory Turnover**：库存效率

**估值方法：**
```
Early: P/GMV = 0.3-0.8x
Growth: P/GMV = 0.2-0.5x
Mature: P/GMV = 0.1-0.3x
或 P/Revenue = 0.5-2x
```

**调整因子：**
- Repeat Rate > 40%: +20-30%
- Repeat Rate < 20%: -20-40%
- Contribution Margin > 15%: +30-50%
- Contribution Margin < 0%: Deal Breaker

---

### 硬科技 / 制造业

**核心指标体系：**
- **Revenue**：收入规模
- **Gross Margin**：40-60%
- **R&D Intensity**：>15%
- **Patent Portfolio**：技术壁垒
- **Backlog**：订单储备

**估值方法：**
```
Pre-Revenue: Berkus, Cost-based
Early Revenue: P/S = 3-6x
Growth: P/E = 20-40x (盈利后)
Mature: P/E = 15-25x
```

**调整因子：**
- 专利 > 50 项：+30-50%
- 毛利率 > 60%: +20-30%
- R&D/Sales > 25%: +10-20%
- Backlog/Sales > 200%: +20-40%

---

### 平台经济 / 双边市场

**核心指标体系：**
- **GMV**：交易规模
- **Take Rate**：5-30%
- **Network Effects**：强度评估
- **Liquidity**：匹配效率
- **Unit Economics**：单客模型

**估值方法：**
```
Early: P/GMV = 0.1-0.3x
Growth: P/Revenue = 3-10x
Mature: P/S = 2-6x
```

**调整因子：**
- 双边网络效应强：+50-100%
- 单边或弱网络效应：-20-40%
- 市场集中度高：+30-50%
- 监管风险高：-30-50%

---

### 医疗健康 / 生物科技

**核心指标体系：**
- **Pipeline Value**：管线估值 (rNPV)
- **Cash Burn**：研发支出
- **Regulatory Milestones**：审批进展
- **Target Market Size**：市场规模
- **Competitive Position**：竞争地位

**临床阶段估值 (rNPV 法)：**
| 阶段 | 估值范围 | 关键假设 |
|------|-------|------|
| Pre-clinical | ¥500-2000 万 | 概念验证 |
| Phase I | ¥2000-5000 万 | 初步安全性 |
| Phase II | ¥5000 万 -2 亿 | 疗效初步数据 |
| Phase III | ¥2-10 亿 | 大规模验证 |
| NDA/BLA Filing | ¥10 亿 + | 接近商业化 |

**调整因子：**
- First-in-class: +100-200%
- Best-in-class: +50-100%
- Fast-follow: 0%
- Safety concerns: -50-80%
- Patent strength: ±50%

---

### 游戏 / 娱乐内容

**核心指标体系：**
- **MAU/DAU**：用户规模
- **ARPPU**：付费用户价值
- **ARPU**：整体价值
- **7/30-day Retention**：留存
- **LTV**：用户生命周期

**估值方法：**
```
Early: P/MAU = ¥50-500/用户
Growth: P/Revenue = 3-8x
Mature: DCF or P/E
```

**调整因子：**
- 30-day Retention > 40%: +30-50%
- 30-day Retention < 20%: -30-50%
- ARPPU > ¥1000/月：+20-40%
- Product pipeline > 3 款：+30-50%
- Single product依赖: -40-60%

## 🎯 估值模型选择决策树

```
项目阶段判断
│
├─ 无收入 (Pre-Seed/Seed)
│   ├─ 仅团队和创意 → Berkus Method
│   ├─ 有原型和早期用户 → Scorecard Valuation
│   └─ 有行业对标 → Early Stage Comps
│
├─ 早期收入 (Seed+/Series A)
│   ├─ SaaS → P/S + NDR 调整
│   ├─ 电商 → P/GMV + 复购率
│   ├─ 硬科技 → P/S + 技术壁垒
│   └─ 其他 → VC Method
│
├─ 成长期 (Series B)
│   ├─ 有清晰单位经济 → DCF + 可比公司
│   ├─ 仍在投入期 → P/S + 增长调整
│   └─ 行业特殊 → 行业专用模型
│
└─ 成熟期 (Series C+)
    ├─ 已盈利 → P/E + EV/EBITDA
    ├─ 接近盈利 → P/S + DCF
    └─ 行业 Leader → Premium Multiple
```

## 🚨 风险识别系统

### 🔴 红色警报 (Deal Breaker)
- 收入确认存在财务造假嫌疑
- 现金流 < 3 个月且无融资计划
- 单一客户 >30% 收入且流失风险高
- 核心创始人法律纠纷
- 核心技术专利无效或侵权风险
- 表外重大负债未披露

### 🟡 黄色警告 (Further Investigation)
- CAC 逐年上升
- 留存率呈下降趋势
- 应收账款周转天数恶化
- 客户集中度 >20%
- 频繁更换审计机构
- 过度资本化研发支出

### 🟢 绿色信号 (Positive Indicators)
- 有机增长占比 >70%
- 经常性收入占比 >85%
- 规模经济效应显现
- 现金流改善趋势
- 财务披露透明度高
